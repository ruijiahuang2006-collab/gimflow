# Demo Audio

This directory contains short Kimusic-generated demonstration audio and/or MIDI examples.

The MP3 samples, when included, were rendered from procedurally generated MIDI using FluidSynth and the FluidR3_GM SoundFont distributed via the Debian/Ubuntu `fluid-soundfont-gm` package. See `../soundfonts/FluidR3_GM_LICENSE.txt` for the full license text and attribution terms.

The baseline retrieval audio library is not redistributed in this repository.
